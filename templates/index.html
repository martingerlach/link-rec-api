Link recommendation to increase the visibility of articles

No guarantees are made that this tool will be maintained. This is an experimental tool hosted on Toolforge. No additional personal data is collected by this tool per the Cloud Services Terms of Use.

There are millions of orphan articles in Wikipedia. These are articles without any incoming links. Thus, orphan articles are not easily accessible by readers when navigating Wikipedia. For more details, see the paper Orphan Articles: The Dark Matter of Wikipedia

This tool aims to increase the visibility of these articles. It generates recommendations of articles from where to link to the given target articles (such as an orphan article). We identify recommendations by looking up the corresponding article in other Wikipedia languages. Surprisingly, in many cases there are already existing links which we can readily "translate". The advantage is that these recommendations can be considered high quality and are easily interpretable because they have already been vetted by other language communities. You can learn more about the model here.

You can test out the tool below with your own article -- for example from the list of orphan articles in English Wikipedia (there are similar lists in other languages). The result is a list of potential source articles from where to link to the orphan article. The suggested link already exists in at least one other Wikipedia.
